Домашнє завдання
Опис завдання: 

Створити проєкт JS.
Підготувати файл promises.js, у якому написати функцію, що відсилатиме за допомогою функції fetch запит на ресурс, який повертає JSON. Через ланцюжок then() обробити цей запит і перевикористати цей JSON у іншій функції.
Сформувати файл async-await.js, що робитиме те саме, що і в promises.js, але цього разу — через підхід async/await.
Зробити файл try-catch.js, у якому надсилатиметься запит на сервіс, якого не існує, і внаслідок помилки запит направлятиметься на той, що існує. Якщо і на іншому ресурсі з якихось причин буде хибна відповідь від сервера —згенерувати власну помилку.
